Registro de Pacientes

POST

  • Atributos:

    name: Deve conter mais de 3 letras.

    susNumber: O número do SUS deve conter 15 caracteres numéricos, só pode ser registrado uma paciente por número. Retorna um 409 caso tente registrar outro. Retorna um 400 caso não contenha os 15 caracteres.

    age: A idade do paciente deve estar entre 0 e 115. Fora desse intervalo retorna um 400.

    priorityTypeCode: Código de prioridade do paciente deve ser preenchido somente com número no intervalo de 0 a 8. Segue a especificação de cada código:

    • 0: SEM PRIORIDADE
    • 1: GESTANTE
    • 2: CRIANCA_COLO
    • 3: LACTANTE
    • 4: AUTISTA
    • 6: DEFICIENTE
    • 7: OBESOS
    • 8: DOADOR_SANGUE
    • O número 5 será preenchido automaticamente levando em consideração a idade do paciente, configurando-o como IDOSO caso possua 60 anos ou mais.

  • Retornará 201 em caso que a persistência de dados seja bem sucedida.
